Spacings

Variable Default value Notice

$spacing-unit-huge floor($spacing-unit-default ($ratio 2))
$spacings

  ‘tight’ 0,
  ‘tiny’ $spacing-unit-tiny,
  ‘small’ $spacing-unit-small,
  ‘large’ $spacing-unit-large,
  ‘huge’ $spacing-unit-huge
)
